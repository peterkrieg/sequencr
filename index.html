Introducing Sequencer

The Image Overlay App

Sequencr is an iOS app.

Creating image overlays is a lot of fun. Express the action of a video into a single image and share it with your friends.

Unfortunately, the current software available for this is very expensive and not user friendly.

There are some simple image sequence apps out there, but they also aren't user friendly
